Яндекс.Музыка - Получение токена

Способ 1: Расширение для браузера

  1. Установите расширение Yandex Music Token для Chrome
  2. Перейдите на сайт music.yandex.ru
  3. Нажмите на иконку расширения в браузере
  4. Скопируйте полученный токен
  5. Вставьте токен в поле выше

Как работает

Токен позволяет приложению получать доступ к вашей библиотеке Яндекс.Музыки: плейлистам, лайкнутым трекам и рекомендациям.

VK - Получение токена

Получение токена

  1. Перейдите на сайт vkhost.github.io
  2. Выберите "VK Audio" или "Kate Mobile"
  3. Нажмите "Разрешить"
  4. Скопируйте ссылку из адресной строки
  5. Токен находится в URL после access_token=

Альтернативный способ

  1. Откройте vkhost.github.io
  2. Нажмите F12 (DevTools)
  3. Перейдите во вкладку Network (Сеть)
  4. Выберите приложение Kate Mobile
  5. Найдите запрос с access_token

Важно

Токен даёт доступ к вашей музыке ВКонтакте. Не передавайте токен третьим лицам.

SoundCloud - Получение токена

Инструкция

  1. Откройте SoundCloud Discover
  2. Нажмите F12 или Ctrl+Shift+I для открытия DevTools
  3. Перейдите во вкладку Network (Сеть)
  4. В поле фильтра введите client_id
  5. Обновите страницу (F5)
  6. Найдите запрос к API SoundCloud
  7. В ответе найдите client_id

Авторизация (опционально)

Для доступа к личной библиотеке:

  1. Войдите в аккаунт SoundCloud
  2. В Network найдите запрос к /connect
  3. Скопируйте значение oauth_token

Без токена SoundCloud работает в ограниченном режиме - только поиск и популярные треки.
